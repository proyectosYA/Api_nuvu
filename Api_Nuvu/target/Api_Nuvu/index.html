Bienvenido

Esta es una sencilla pagina html que consume un servicio rest por medio de javascript y su objeto XMLHttpResquest. El area de usuarios hace una peticion al recurso y por medio de una tabla dinamica muestra los resultados obtenidos desde el servidor en su formato json. El framework css que se utilizo es Materialize.

Consideraciones:

* La parte backend en java esta funcional, para consumir con un cliente rest como postman u otro.

RECURSOS:
http://localhost:8082/Api_Nuvu/api/signin -------> login, registro y retorna un token

http://localhost:8082/Api_Nuvu/api/usuarios -------> todos los usuarios
http://localhost:8082/Api_Nuvu/api/usuarios/34543213 -------> usuario por cedula
http://localhost:8082/Api_Nuvu/api/usuarios/agregar -------> agrega usuario
http://localhost:8082/Api_Nuvu/api/usuarios/modficar -------> modifica usuario


http://localhost:8082/Api_Nuvu/api/tarjetas -------> todos los usuarios con tarjeta
http://localhost:8082/Api_Nuvu/api/tarjetas/34543213 ------->por cedula
http://localhost:8082/Api_Nuvu/api/tarjetas/agregar -------> agrega tarjeta
http://localhost:8082/Api_Nuvu/api/tarjetas/modficar -------> modifica


*La parte FRONT tiene cuatro paginas html que consumen el api.
